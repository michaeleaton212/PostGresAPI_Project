# Startanleitung

## Backend starten
```bash
cd \PostGresAPI_Project\PostGresAPI_Project\PostGresAPI
dotnet run
```

## Frontend bauen und lokalen i18n-Server starten
```bash
cd \PostGresAPI_Project\PostGresAPI_Project\Frontend\frontend\my-app
ng build
node local-i18n-server.mjs
```
